# RICHE RACS Website

A simple static website styled to match the provided reference image.

## Features

- Sticky logo header and navigation
- Hero banner with collection text
- Best sellers section with prices in USD
- Separate pages for Hoodies, Pants, Accessories, and Contact
- Footer with policies and social links

## How to View

Open `index.html` in your web browser and use the header links to navigate to the separate category pages.

## Notes

The page uses placeholder text for the product names and descriptions so you can easily replace them with your own content.
